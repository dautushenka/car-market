Использование » Работа с дополнительными полями
После создания дополнительного поля в админке модуля в шаблонах можно использовать следующие теги
{xfield_1_title} - Название поля с ID 1;
{xfield_1_descr} - Описание поля с ID 1;
{xfield_1_value} - Значение поля с ID 1. Используется в шаблонах отображения авто;
{xfield_1_html} - Форма для заполения поля с ID 1. Используется в шаблоне добавления авто;

Тег {xfields} Выводит все поля сразу, по полю в строке. Формат
{xfield_1_title}: {xfield_1_html}
или
{xfield_1_title}: {xfield_1_value}


Также в шаблоне отображения авто можно использовать блок
[xfield_1][/xfield_1]
в кторый можно заключить текс, который будет выводиться только если поле было заполенно
