Car Market (Авторынок) v2.3.1
Установка
1. Заливаем все файлы из папки uploads на сервер, они имеют такую же сткрутуру каталогов как и DLE.
2. Содержимое папки templates (файлы *.tpl) скопировать в папку с вашим с шаблоном, зарания их подогнать под ваш шаблон, по умолчанию они сделаны для шаблона Default.
3. Установите файл /engine/car-market/auto_cron.php на запуск крона, примеорно 3-4 раза в день.
4. Зарустить мастер установки http://www.ваш_домен/install.php, далее следуйте его инструкциям. В зависимости выбора опций при установки с помощью мастера будут происходить изменения далее следовательно не переходите к следующим пункам пока не закончите работу с мастером установки.
