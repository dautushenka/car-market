4. Как сделать, что бы при клике на фото открывался полный обзор авто
Открываем файл /engine/car-market/user/default.php
Найдём строку 142
$array = $auto->ShowAuto($id); 

И заменяем ее на такую
$array = $auto->ShowAuto($id, array("show_photo" => 0)) + ShowPhoto($id, $auto); 


Те же самые действия в принципе только в файле main.php
Найти строка 120
$template->SetRow($auto->ShowAuto($id), "row_auto"); 

заменить на
$template->SetRow($auto->ShowAuto($id, array("show_photo" => 0)) + ShowPhoto($id, $auto), "row_auto");
